## How to evaluate?

每个人的申请总结，只是个人经验。观点不一定全对，经验也不一定适用其他人。每年申请形势都有变化、有差异,今年的录取政策、申请的难易程度和去年可能不一样。 影响申请结果的因素很多，不要简单地按硬件条件对号入座。美国大学有可能拒掉成绩高的而录取成绩低的，靠成绩搞定的学校，也可能是你不想的。
